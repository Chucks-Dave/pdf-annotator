## Installation

Follow these steps to install and set up your Next.js project:

1. Clone the repository:
    ```sh
    git clone https://github.com/Chucks-Dave/pdf-annotator.git
    cd pdf-annotator
    ```

2. Install dependencies:
    ```sh
    npm install
    ```

3. Start the development server:
    ```sh
    npm run dev
    ```
